# NFR 프로파일

## 프로파일 코드: B-A-B-C-B
- **속도 (A)**: < 1초 목표. STT 변환 및 Gemini 추론 시간을 최소화해야 함.
- **정확성 (B)**: 팩트 체크 오류 허용. 최종 검증은 사람이 수행.
- **보안 (C)**: 표준 TLS 암호화 및 OAuth 인증.

## 식별된 충돌 (Conflict)
⚠️ **속도(A) vs 정확성(B)**: 
Gemini가 정확한 분석을 하려면 긴 컨텍스트를 읽어야 하는데, 이는 응답 속도를 늦춤. 
→ **Stage 2에서 트레이드오프 결정 필요 (예: 스트리밍 답변 사용)**
